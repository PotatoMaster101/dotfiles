# Dot Files
Collection of my dotfiles.

## Setup (\*nix)
Run in Bash:
```
$ git clone https://github.com/PotatoMaster101/dotfiles
$ cd ./dotfiles && bin/sh ./init.sh
```

## Setup (Windows)
Run in PowerShell:
```
$ git clone https://github.com/PotatoMaster101/dotfiles
$ .\dotfiles\init.ps1
```
